# Atlas CLI

```
python3 atlas/cli/atlas.py init
python3 atlas/cli/atlas.py status --control-plane http://localhost:8080
python3 atlas/cli/atlas.py devices --control-plane http://localhost:8080
python3 atlas/cli/atlas.py logs tail --log atlas/logs/ai.jsonl
```

## Auth
- Provide `--token` or set `ATLAS_API_TOKEN`.
